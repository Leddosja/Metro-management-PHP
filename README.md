# Metro Management System - PHP Web Application

Progetto scolastico sviluppato con l’obiettivo di realizzare un sistema web per la gestione di una metropolitana, includendo autenticazione utenti, prenotazioni e gestione dei dati tramite database relazionale.

## 📖 Descrizione

L’applicazione è una web app sviluppata in PHP che permette agli utenti di registrarsi, effettuare il login e gestire prenotazioni all’interno di un sistema di trasporto metropolitano.

Il progetto include la progettazione e implementazione di un database MySQL, con tabelle relazionali per utenti, stazioni, linee, treni e prenotazioni.

L’obiettivo principale è stato comprendere lo sviluppo di applicazioni web dinamiche e l’integrazione con database relazionali.

> ⚠️ Questo progetto è stato sviluppato a scopo formativo e potrebbe non seguire completamente le best practice moderne.

## 🚀 Funzionalità principali

- Registrazione e autenticazione utenti
- Sistema di login con gestione sessioni
- Prenotazione di viaggi/metropolitana
- Gestione dati tramite database MySQL
- Query SQL per operazioni CRUD
- Navigazione tra diverse pagine (login, ricerca, prenotazioni)

## 🛠️ Tech Stack

- **Linguaggio:** PHP  
- **Frontend:** HTML, CSS  
- **Database:** MySQL  
- **Strumenti:** MySQL Workbench, VS Code  

## 🗄️ Database

Il progetto include script SQL per la creazione delle tabelle:

- Utenti  
- Stazioni  
- Linee  
- Treni  
- Prenotazioni  

## 📌 Note

Progetto realizzato a scopo formativo.  
Il codice rappresenta una prima esperienza nello sviluppo web e nella gestione di database relazionali.
